# 安装指南

Chat2Note 适用于 Chrome、Firefox 和其他基于 Chromium 的浏览器。按照以下说明在您的浏览器中安装。

## Chrome / Edge / Brave

### 从 Chrome 网上应用店安装（推荐）

1. 访问 [Chrome 网上应用店中的 Chat2Note 页面](#)（即将推出）
2. 点击**添加至 Chrome**（或添加至 Edge/Brave）
3. 在提示时确认安装
4. Chat2Note 图标将出现在您的浏览器工具栏中

### 从源代码安装（开发者模式）

如果您想使用最新的开发版本：

1. 从 [GitHub Releases](https://github.com/Chat2Note/releases) 下载最新版本
2. 将 ZIP 文件解压到一个文件夹
3. 打开浏览器并导航至：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
4. 启用**开发者模式**（右上角的开关）
5. 点击**加载已解压的扩展程序**
6. 选择解压后的文件夹
7. Chat2Note 现已安装！

## Firefox

### 从 Firefox 附加组件安装（推荐）

1. 访问 [Firefox 附加组件中的 Chat2Note 页面](#)（即将推出）
2. 点击**添加到 Firefox**
3. 在提示时确认安装
4. Chat2Note 图标将出现在您的浏览器工具栏中

## 验证安装

安装后，您应该看到：

1. 浏览器工具栏中的 Chat2Note 图标
2. 当您访问支持的 AI 聊天平台（ChatGPT、Claude、DeepSeek、Gemini、Kimi、腾讯元宝、豆包、Grok 等）时，您将看到导出按钮出现

## 下一步

现在 Chat2Note 已安装，在[快速开始指南](/zh-cn/getting-started/quick-start)中学习如何使用它。

## 权限说明

Chat2Note 需要以下权限：

- **activeTab**：访问当前标签页的内容并提取对话
- **downloads**：将导出的文件保存到您的计算机
- **storage**：保存您的偏好设置
- **clipboardWrite**：将导出的内容复制到剪贴板
- **主机权限**：在支持的 AI 聊天平台上运行

所有处理都在本地进行 - 不会将数据发送到外部服务器。
