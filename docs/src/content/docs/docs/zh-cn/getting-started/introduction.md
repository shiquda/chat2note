# 欢迎使用 Chat2Note

Chat2Note 是一个**注重隐私**的浏览器扩展，可让您将AI聊天对话导出为多种格式和目标位置。通过完全本地处理且不传输任何外部数据，您的对话始终保持私密和安全。

## 什么是 Chat2Note？

Chat2Note 从流行的AI聊天平台中提取对话，并将其导出为各种格式：

- **Markdown** - 保留完整格式
- **JSON** - 结构化数据格式
- **纯文本** - 简单文本导出

## 核心功能

### 🔒 隐私优先

所有处理都在您的浏览器本地进行。数据永远不会发送到外部服务器。

### 🌐 多平台支持

目前支持：

- ChatGPT (chatgpt.com)
- Claude (claude.ai)
- DeepSeek (chat.deepseek.com)
- Gemini (gemini.google.com)
- Kimi (kimi.com)
- 腾讯元宝 Yuanbao (yuanbao.tencent.com)
- 豆包 Doubao (doubao.com)
- Grok (grok.com)

### 📤 多种导出目标

- **本地文件** – 下载 Markdown、JSON 或 TXT 格式
- **Notion** – 将对话作为结构化页面写入数据库
- **Obsidian** – 通过 `obsidian://` URI 保存到本地仓库
- **思源笔记 (SiYuan)** – 将 Markdown 文档写入本地笔记本
- **Joplin** – 通过 Web Clipper API 创建笔记并附加标签
- **剪贴板** – 复制排版后的内容用于快速粘贴

### ✨ 智能功能

- **消息选择** – 精确导出指定片段（_Pro 会员专享_）
- **角色过滤** – 仅导出用户或助手消息
- **元数据控制** – 附加时间戳、来源链接、平台名称（_Pro 会员专享_）
- **自定义 Markdown 模板** – 定义对话与单条消息的渲染方式（_Pro 会员专享_）
- **Notion 属性映射** – 映射来源链接、导出时间、平台、消息数量
- **配置备份与恢复** – 一键导出/导入全部设置
- **悬浮按钮位置** – 在页面上移动或隐藏导出入口

## 为什么选择 Chat2Note？

管理AI对话对以下用途至关重要：

- **知识管理** – 借助 Notion、Obsidian、SiYuan、Joplin 构建个人知识库
- **内容创作** – 快速生成可直接发布的 Markdown 草稿
- **团队协作** – 在共享数据库或同步仓库中协同复盘
- **研究工作流** – 追踪问题、回答与参考链接
- **会议记录** – 自动整理议程、决策与待办
- **PKM（个人知识管理）** – 结构化沉淀长期可检索的信息

## Pro 会员权益

升级 Chat2Note Pro，可解锁进阶工作流：

- **无限导出** – 移除每日导出次数限制
- **选择性导出** – 支持片段/单条消息导出
- **丰富元数据** – 添加时间、来源、模型等额外字段
- **自定义模板** – 控制文件名、整体结构与消息渲染
- **增强集成** – 开启 Obsidian YAML Frontmatter 等高级选项
