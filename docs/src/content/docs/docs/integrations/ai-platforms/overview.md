# AI Platform Integrations

Chat2Note integrates seamlessly with popular AI chat platforms, allowing you to export conversations directly from your browser.

## Primary Platforms

### ChatGPT

**Website:** [chat.openai.com](https://chat.openai.com) | [chatgpt.com](https://chatgpt.com)

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Code blocks with syntax highlighting
- ✅ Markdown formatting
- ✅ Message timestamps
- ⚠️ Images not fully preserved

**Requirements:**

- ChatGPT account (free or Plus)
- Chrome, Firefox, Edge, or Brave browser

### Claude

**Website:** [claude.ai](https://claude.ai)

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Markdown formatting
- ✅ Code blocks
- ✅ Message timestamps
- ✅ Conversation metadata

**Requirements:**

- Claude account
- Chrome, Firefox, Edge, or Brave browser

### DeepSeek

**Website:** [chat.deepseek.com](https://chat.deepseek.com)

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Code blocks
- ✅ Markdown formatting
- ✅ Message timestamps

**Requirements:**

- DeepSeek account
- Chrome, Firefox, Edge, or Brave browser

### Gemini

**Website:** [gemini.google.com](https://gemini.google.com)

## How It Works

1. **Install Chat2Note** in your browser
2. **Visit any supported platform** and start a conversation
3. **Click the export button** that appears on the page
4. **Choose your export format** and destination
5. **Export your conversation** with one click
