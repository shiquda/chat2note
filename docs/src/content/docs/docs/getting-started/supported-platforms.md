# Supported Platforms

Chat2Note currently supports the following AI chat platforms. Each platform has been carefully tested to ensure reliable conversation extraction.

## Primary Platforms

These are the globally popular AI chat experiences that most of our users rely on day-to-day. They receive the most polish and ongoing compatibility updates.

### ChatGPT

**URLs:**

- `https://chat.openai.com`
- `https://chatgpt.com`

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Code blocks with syntax highlighting
- ✅ Markdown formatting
- ✅ Message timestamps
- ⚠️ Images not fully preserved

### Claude

**URL:** `https://claude.ai`

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Markdown formatting
- ✅ Code blocks
- ✅ Message timestamps
- ✅ Conversation metadata

### DeepSeek

**URL:** `https://chat.deepseek.com`

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Code blocks
- ✅ Markdown formatting
- ✅ Message timestamps

### Gemini

**URL:** `https://gemini.google.com`

**Features:**

- ✅ Full conversation history
- ✅ User and assistant messages
- ✅ Rich content conversion
- ✅ Code blocks
- ⚠️ LaTeX formulas may have issues

## Regional & Emerging Platforms

We also offer full support for fast-growing platforms popular in Asia and the xAI ecosystem. These integrations share the same export experience and benefit from the metadata, formatting, and automation features provided by Chat2Note.

### Kimi

**URL:** `https://kimi.com`

**Highlights:**

- ✅ API-based extraction for long conversations
- ✅ Preserves message order and timestamps
- ✅ Ideal for Moonshot AI users working primarily in Chinese

### Yuanbao

**URL:** `https://yuanbao.tencent.com`

**Highlights:**

- ✅ Captures enterprise-grade chat history from Tencent Yuanbao
- ✅ Includes sender roles and structured message content
- ✅ Works with authenticated sessions via secure API calls

### Doubao

**URL:** `https://doubao.com`

**Highlights:**

- ✅ Designed for ByteDance's Doubao workspace
- ✅ Handles threaded replies and assistant suggestions
- ✅ Great for productivity workflows within Feishu/Lark ecosystems

### Grok

**URL:** `https://grok.com`

**Highlights:**

- ✅ Extracts real-time conversations from xAI's Grok interface
- ✅ Supports metadata retrieval when available
- ✅ Useful for researchers relying on Grok's up-to-date knowledge
