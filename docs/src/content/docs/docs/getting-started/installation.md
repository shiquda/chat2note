# Installation Guide

Chat2Note is available for Chrome, Firefox, and other Chromium-based browsers. Follow the instructions below to install it in your browser.

## Chrome / Edge / Brave

### Install from Chrome Web Store (Recommended)

1. Visit the [Chat2Note page on Chrome Web Store](#) (coming soon)
2. Click **Add to Chrome** (or Add to Edge/Brave)
3. Confirm the installation when prompted
4. The Chat2Note icon will appear in your browser toolbar

## Firefox

### Install from Firefox Add-ons (Recommended)

1. Visit the [Chat2Note page on Firefox Add-ons](#) (coming soon)
2. Click **Add to Firefox**
3. Confirm the installation when prompted
4. The Chat2Note icon will appear in your browser toolbar

## Verify Installation

After installation, you should see:

1. The Chat2Note icon in your browser toolbar
2. When you visit a supported AI chat platform (ChatGPT, Claude, DeepSeek, Gemini, Kimi, Yuanbao, Doubao, Grok, etc.), you'll see an export button appear

## Next Steps

Now that Chat2Note is installed, learn how to use it in the [Quick Start Guide](/getting-started/quick-start).

## Permissions Explained

Chat2Note requires the following permissions:

- **activeTab**: To access the current tab's content and extract conversations
- **downloads**: To save exported files to your computer
- **storage**: To save your preferences and settings
- **clipboardWrite**: To copy exported content to clipboard
- **Host permissions**: To run on supported AI chat platforms

All processing happens locally - no data is sent to external servers.
